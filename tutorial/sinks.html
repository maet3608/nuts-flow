Writing to Sinks¶

Iterables¶

All Python functions that accept iterators/iterables can serves as sinks, e.g.

list, set, dictionary, file.writelines, sum, min, max

with open(filepath) as f:
  f.writelines(Range(5) >> Square() >> Map(str))
